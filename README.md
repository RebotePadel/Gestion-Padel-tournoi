# Gestion Padel Tournoi

## Télécharger le fichier `Tournoi-V4.html`

1. Ouvrez le dépôt GitHub dans votre navigateur.
2. Cliquez sur le fichier **Tournoi-V4.html** dans la liste des fichiers.
3. Dans la vue du fichier, cliquez sur le bouton **Raw** (ou **Brut**).
4. Faites un clic droit dans la page affichée puis sélectionnez **Enregistrer sous...** pour sauvegarder le fichier HTML localement.
5. Ouvrez ce fichier en local dans votre navigateur pour utiliser l'application.

### Alternative en ligne de commande (si vous avez l'URL du dépôt)

```bash
# Clone du dépôt
# git clone <URL_DU_DEPOT>
# cd Gestion-Padel-tournoi
# Récupération directe du fichier (remplacez par l'URL raw exacte)
# curl -L -o Tournoi-V4.html "https://raw.githubusercontent.com/<organisation>/<repo>/branch/Tournoi-V4.html"
```

Remplacez `<URL_DU_DEPOT>` et l'URL `raw` par celles de votre dépôt GitHub.
